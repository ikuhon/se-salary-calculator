# SE年収診断ツール

システムエンジニアの適正年収を診断するWebツールです。

## 機能
- 経験年数・スキル・地域から適正年収を算出
- 詳細な分析とアドバイス提供
- SNS共有機能
- モバイル対応

## デモ
https://yourusername.github.io/se-salary-calculator/

## 使い方
1. 基本情報（経験年数・働き方）を入力
2. 技術スキル・開発言語を選択
3. 勤務地・業界を選択
4. 診断結果と改善アドバイスを確認

## 技術仕様
- Pure HTML/CSS/JavaScript
- レスポンシブデザイン
- GitHub Pages対応
